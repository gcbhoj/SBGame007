Welcome to Memory Game

This is a memory card matching game that helps improve your memory and concentration.

Description

The objective of the game is to find matching pairs of cards. As you progress, your memory and attention to detail will improve.

How to Play

  • Click on a card to reveal its image.
  • Match the image card with the corresponding word card.
  • If the match is correct, both cards will stay revealed.
  • If the match is incorrect, both cards will flip back after a brief delay.
  • Try to match all pairs with the least number of wrong moves and in the shortest time.
  • Once all pairs are matched, you win the game!
